📌 Passo 1 — Transformar a base em Tabela
1.	Abra a aba Bases (onde estão os dados dos clientes).
2.	Selecione toda a tabela (incluindo cabeçalhos).
3.	Pressione Ctrl + T e confirme que “Minha tabela tem cabeçalhos” está marcado.
👉 Isso facilita a atualização automática e o uso em tabelas dinâmicas.
________________________________________
📌 Passo 2 — Inserir Tabelas Dinâmicas
1.	Vá em Inserir > Tabela Dinâmica.
2.	Escolha a tabela criada como origem.
3.	Crie uma nova aba chamada Dashboard e insira a tabela dinâmica lá.
4.	Repita o processo para criar várias tabelas dinâmicas, por exemplo: 
o	Receita total por Plano (Ultimate, Standard, Core).
o	Receita por Tipo de Assinatura (Mensal, Anual, Trimestral).
o	Receita por Mês (usando Start Date).
o	Receita com passes extras (EA Play, Minecraft).
________________________________________
📌 Passo 3 — Criar Gráficos Dinâmicos
1.	Clique dentro da tabela dinâmica.
2.	Vá em Inserir > Gráfico Dinâmico.
3.	Escolha o tipo de gráfico adequado: 
o	Colunas/Barras → comparar planos.
o	Pizza/Donut → participação de cada tipo de assinatura.
o	Linha → evolução mensal da receita.
4.	Posicione os gráficos na aba Dashboard.
________________________________________
📌 Passo 4 — Adicionar Segmentações de Dados (Slicers)
1.	Clique na tabela dinâmica.
2.	Vá em Inserir > Segmentação de Dados.
3.	Selecione campos como Plano, Tipo de Assinatura ou Auto Renewal.
4.	Posicione os filtros no dashboard.
👉 Isso permite interatividade: você clica no filtro e todos os gráficos se atualizam.
________________________________________
📌 Passo 5 — Criar Indicadores (KPIs)
1.	Insira Cartões de Texto ou use Gráficos de Indicador para mostrar: 
o	Receita total.
o	Número de assinantes ativos.
o	Ticket médio (Receita ÷ Assinantes).
2.	Use fórmulas simples como =SOMA([Total Value]) em células auxiliares e destaque no dashboard.
________________________________________
📌 Passo 6 — Aplicar Paleta de Cores
No seu arquivo já há uma paleta definida (#9BC848, #22C55E, #2AE6B1, etc.).
•	Use verde para Ultimate, azul para Standard, cinza para Core.
•	Aplique essas cores nos gráficos e nos slicers para manter identidade visual.
________________________________________
👉 Resultado: um painel interativo e visual, onde você consegue analisar receita por plano, evolução mensal, impacto dos passes extras e participação dos tipos de assinatura.
