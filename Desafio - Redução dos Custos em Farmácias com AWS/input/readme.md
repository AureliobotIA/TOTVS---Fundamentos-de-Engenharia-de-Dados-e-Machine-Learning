## 📌 Introdução

Este relatório apresenta o processo de implementação de ferramentas AWS em uma farmácia, com foco na **redução de custos operacionais** e aumento da eficiência. O objetivo foi selecionar 3 serviços AWS que proporcionassem economia imediata e escalabilidade para o negócio.

## 📉 Conclusão

A implementação dos serviços AWS na farmácia trouxe benefícios imediatos:
- Redução de custos com servidores locais.  
- Automação de processos repetitivos.  
- Melhor visualização dos dados para tomada de decisão.  

Recomenda-se a continuidade do uso das ferramentas e a exploração de novos serviços AWS para ampliar ainda mais a eficiência.
